# API Reference v2.4 ## Authentication To access the Project Alpha repository, you must provide a valid Bearer token in the header of each request. ```json { "status": "success", "endpoint": "/v2/auth/verify", "timestamp": "2023-11-24T12:00:00Z" } ``` ## Endpoints - GET /archive/search - POST /archive/ingest - DELETE /archive/purge Note: All requests are logged for security auditing purposes.

API Reference v2.4

Authentication

To access the Project Alpha repository, you must provide a valid Bearer token in the header of each request. 

{
  "status": "success",
  "endpoint": "/v2/auth/verify",
  "timestamp": "2023-11-24T12:00:00Z"
}

Endpoints

  • GET /archive/search
  • POST /archive/ingest
  • DELETE /archive/purge

Note: All requests are logged for security auditing purposes.
