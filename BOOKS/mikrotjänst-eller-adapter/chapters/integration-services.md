# 4. Integration services

**Syfte:** Koppla system till varandra.

Detta är exakt den typ du beskrev.

Exempel:

```
SAP Customer AdapterPayment Provider AdapterCRM Integration
```

De gör ofta:

- datamappning
- protokollöversättning
- autentisering
- API-anpassning

De representerar **inte en verksamhetsförmåga**.
