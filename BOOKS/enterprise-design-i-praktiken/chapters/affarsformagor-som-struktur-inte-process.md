# 2. Affärsförmågor som struktur – inte process

> Chapter-ID: affarsformagor-som-struktur-inte-process
> Status: draft

Affärsförmågor beskriver vad organisationen måste kunna, oberoende av hur det realiseras tekniskt.

I praktiken används affärsförmågor som ett sätt att skapa sammanhang för krav, tjänster och system.

Exempel på vanliga affärsförmågor är Kundhantering, Orderläggning, Leverans och Betalning.

Förslag till illustration: En karta med affärsförmågor som stabil grund, med föränderliga lösningar under.
