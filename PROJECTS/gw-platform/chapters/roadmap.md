# Roadmap

> Chapter-ID: roadmap
> Status: draft

## Fas 1: Grundplattform (avslutad)

- Kapitelhantering med markdown och metadata
- Webbviewer med sidebar-navigation
- Word-import och PDF-generering
- File watcher for automatisk statusuppdatering

## Fas 2: Multi-bok och browsing (pagaende)

- Stod for flera bocker i samma REPOSITORY
- Miller columns file browser
- BOOKS/PROJECTS-separation med runtime-vaxling
- Inline-redigering i sidebar

## Fas 3: AI-agentintegration (planerad)

- MCP-integration for agentaccess till kapiteldata
- Automatisk begreppsverifiering
- Dependency chain mellan kapitel
- Smarta samlingar baserat pa metadata
